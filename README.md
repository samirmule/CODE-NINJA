# 🍽️ HungerBridge

## 📌 Project Description

HungerBridge is a real-time food donation and distribution platform that connects donors (restaurants, wedding halls, canteens) with NGOs and volunteers to ensure surplus food reaches those in need within a short time window.

The system helps reduce food waste and solve hunger issues by enabling quick coordination and efficient delivery.

---

## ❗ Problem Statement

Every day, large amounts of food are wasted while many people go hungry. There is no efficient system to connect food donors with NGOs and volunteers in real-time, resulting in missed opportunities to utilize surplus food.

---

## 🚀 Features & Functionality

* 🍱 Donor can post surplus food (name, quantity, location)
* ⏳ Real-time expiry tracking (2-hour window)
* 📋 NGOs can view available food listings
* 🤝 Claim system to reserve food
* 🚚 Volunteer delivery dashboard
* 📊 Status tracking (Posted → Claimed → Delivered)
* 📍 Simple and user-friendly UI for fast interaction
